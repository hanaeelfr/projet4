Compter le nombre de voyelles dans une chaine

Enoncé

Ecrire un programme Javascript qui permet de calculer le nombre de voyelles (a,e,i,o,u,y) dans une phrase saisie par l'utilisateur.

Démo

Code source

function compter(){
var str =document.getElementById('text').value;
var voyelles ="aeiouyAEIOUY";
var ctr=0;
for(var x=0;x for(var i=0;i if(str[x]==voyelles[i])
{
ctr++;
}
}
document.getElementById("resultat").innerHTML="the number of vowel is "+ctr;
}
}
