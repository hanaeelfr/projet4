Pyramid triangle star pattern - javascript

Losange en étoiles

Enoncé

Ce programme a pour objectif de dessiner un losange en étoiles dont la longueur des diagonales est un nombre impair saisi par l'utilsateur.
Dans le cas où l'utilisateur entre un nombre pair, il faut afficher une erreur.

Code Source

function dessinEtoile() {
// print stars in Diamond-shaped pattern
var etoile = document.getElementById("etoile").value;
var para = document.getElementById("para");
var i, k;
for (i = 1; i <= etoile; i++) {
for (k = 1; k <= i; k++) {
para.innerHTML += "*";
}
para.innerHTML += "
";
}
}
